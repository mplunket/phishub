## phishub

The #1 Site for Musicians Who Love Phish

## Open-Source

This site is open source on [Github](https://github.com/mplunket/phishub) and voluntary complies with [the Phish fan web site policy](http://www.phish.com/faq/web-guidelines).

## Features

- Songs
  - A searchable database of Phish's entire repertoire
  - Filter by Starting Letter (A-Z), Original vs. Cover, Original Artist
  - For each song, view all associated Tabs, Lyrics, Performances, Lessons, and Chat
- Tabs
  - Guitar, Bass, and Keys/Piano tabs
  - Format???
  - Chat
- Performances
  - Links to Phish (official / unofficial) on YouTube
  - Links to Cover Band/Individual performances on YouTube
  - User-uploaded videos?
- Lessons
  - Lessons from YouTube
  - User-uploaded videos?
- Setlists
  - Create multiple song lists
  - Share song lists with other users
- Chat
  - Embed Chat in various areas of the platform -- Tabs, Songs, Performances, Lessons, etc.
  - Ability to @tag other users
